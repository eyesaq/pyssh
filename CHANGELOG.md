# Changelog
All notable changes to **pyssh** will be documented in this file.

---

## [1.0.0] - 2024-02-01
### Added
- Initial stable release of **pyssh**.
- Core SSH connection handling using Paramiko.
- Remote command execution with stdout/stderr capture.
- Basic GUI framework built with CustomTkinter.
- Project structure including `app/`, `data/`, and `main.py`.
- Pillow integration for future image-handling features.
- `todo.txt` outlining planned enhancements.

### Notes
- This release establishes the foundation for future development
