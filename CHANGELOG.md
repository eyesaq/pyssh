### Notes

## [1.1.0-beta] - xx-xx-xxxx
### Changes
- Improved GUI.
- Added the ability to refresh connections on-demand.
- Improved logging capabilities.
- Added `config.py` for developers.
- Added PNG icons.
- Added many keyboard shortcuts to enable keyboard navigation.
- Added tooltips.
- Improved connection management validation.
- Threaded long background processes.
- Changed the changelog's history.
- Fixed many bugs.
- Removed Herobrine.

### Notes
- There is still no in-window SSH console.
